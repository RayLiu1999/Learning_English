# 📚 不熟單字庫與測驗系統

這個目錄包含個人不熟悉單字的記錄與練習測驗。

## 📂 目錄結構

```
vocabulary/
├── README.md           # 本說明文件
├── words.md            # 單字庫（所有單字的主檔案）
├── progress.md         # 學習進度追蹤
├── mistakes.md         # 錯誤單字追蹤
└── quizzes/            # 測驗題目資料夾
    ├── quiz-001.md
    └── ...
```

---

## 🎯 使用流程

### Step 1: 做測驗

打開 `quizzes/` 裡的測驗檔案，作答完畢後記錄你的答案。

### Step 2: 提交答案

把你的答案貼給 AI 助理，格式如下：

```
Quiz-001 答案：
Q1: A
Q2: C
Q3: B
...
```

### Step 3: 獲得批改結果

AI 會：

1. ✅ 批改你的答案，告訴你對錯
2. 📊 更新 `progress.md` 測驗紀錄
3. ❌ 更新 `mistakes.md` 錯誤單字追蹤
4. 📝 更新 `words.md` 單字的測驗狀態

---

## 📝 單字格式說明

| 欄位     | 說明                       |
| -------- | -------------------------- |
| 單字     | 英文單字或片語             |
| 中文意思 | 中文翻譯與解釋             |
| 詞性     | 名詞、動詞、形容詞、副詞等 |
| 重要程度 | 1-5 分（5 分最重要）       |

---

## ⭐ 重要程度說明

| 分數  | 說明               | 出題規則                     |
| ----- | ------------------ | ---------------------------- |
| **5** | 核心單字，必須熟練 | 高頻出題、多種題型、優先練習 |
| **4** | 重要單字，經常使用 | 中高頻出題、多種題型         |
| **3** | 實用單字，偶爾使用 | 中頻出題、基礎題型           |
| **2** | 補充單字，了解即可 | 低頻出題、簡單題型           |
| **1** | 額外單字，認識就好 | 偶爾出現、認識性題目         |

---

## 📋 題目類型

| 題型             | 說明                         |
| ---------------- | ---------------------------- |
| 選擇題（英翻中） | 根據英文選出正確的中文意思   |
| 選擇題（中翻英） | 根據中文選出正確的英文單字   |
| 填空題           | 根據句子上下文填入正確的單字 |
| 配對題           | 將單字與其定義進行配對       |
| 情境應用題       | 在模擬職場情境中選擇適當單字 |

---

## 📊 出題規則

- **重要程度 5**：每次測驗必出，至少 2-3 題，包含多種題型
- **重要程度 4**：高機率出題，1-2 題
- **重要程度 3**：中等機率出題
- **重要程度 2-1**：低機率出題，作為補充
- **錯誤率高的單字**：優先出題加強練習

每份測驗建議 15-25 題，混合各種題型與難度。

---

## 📈 錯誤追蹤機制

`mistakes.md` 會記錄：

- 每個單字的錯誤次數
- 最近錯誤日期
- 容易混淆的選項
- 錯誤率統計

這些資訊會用於：

1. 優先出高錯誤率單字的題目
2. 設計針對混淆點的題目
3. 追蹤學習進步情況
