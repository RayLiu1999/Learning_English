# 職場英文測驗系統 Workflow

本 workflow 定義了職場英文測驗系統的完整操作流程。

## 📂 相關檔案位置

- `workplace/` — 學習資料（19 個主題）
- `workplace-question/` — 測驗與追蹤
  - `progress.md` — 學習進度追蹤
  - `mistakes.md` — 錯誤項目追蹤
  - `[topic]/quiz-XXX.md` — 各主題測驗（每個主題一個資料夾）
  - `mixed/` — 混合測驗目錄

---

## 🎯 任務一：進行主題測驗

當用戶要求做某主題測驗時：

### Step 1: 確認測驗檔案

// turbo

1. 確認主題對應的測驗檔案存在
2. 提示用戶開始作答

### Step 2: 接收答案

用戶格式：

```
[topic]-quiz 答案：
1: B
2: C
3: B
...
```

### Step 3: 批改並計算成績

1. 對照答案區批改每一題
2. 計算：正確題數、錯誤題數、正確率
3. 區分各題型表現（Vocabulary / Phrase / Grammar）

### Step 4: 回覆批改結果

向用戶顯示：

- 成績統計
- 各題型表現
- 錯誤題目清單（題號、類型、你的答案、正確答案）
- 重點片語/句型說明

### Step 5: 更新測驗檔案

在該測驗檔案末尾加入「作答紀錄」區塊：

```markdown
---

## 📝 作答紀錄

### [日期] 第 X 次測驗

| 項目   | 結果  |
| ------ | ----- |
| 正確   | XX/30 |
| 正確率 | XX%   |

#### 各題型表現

| 題型       | 正確/總數 |
| ---------- | --------- |
| Vocabulary | X/10      |
| Phrase     | X/10      |
| Grammar    | X/10      |

#### 錯誤題目

| 題號 | 類型   | 我的答案 | 正確答案         |
| ---- | ------ | -------- | ---------------- |
| X    | Phrase | A        | B (正確用法說明) |
```

### Step 6: 更新 progress.md

1. 更新該主題的測驗次數、最高分、平均分
2. 更新狀態（⬜ → 🟡 → 🟢）
3. 更新題型表現統計
4. 新增測驗紀錄

### Step 7: 更新 mistakes.md

1. 將錯誤項目加入對應區塊
2. 如有混淆組合，加入記錄
3. 更新錯誤來源追蹤

---

## 🎯 任務二：產生混合測驗

當用戶要求產生混合測驗時：

### Step 1: 確認測驗編號

// turbo

1. 查看 `custom-quizzes/` 目錄
2. 新編號 = 最新編號 + 1（格式：mixed-001, mixed-002...）

### Step 2: 選擇出題來源

1. 讀取 `mistakes.md` 取得錯誤項目
2. 讀取各主題測驗，選擇題目
3. 優先順序：
   - **最優先**：錯誤率高的項目
   - **次優先**：未測驗過的主題
   - **一般**：已測驗但需複習的主題

### Step 3: 產生測驗

建立 `custom-quizzes/mixed-XXX.md`：

```markdown
# 📝 Mixed Quiz XXX

> Date: YYYY-MM-DD
> Questions: 20-25
> Topics: [列出涵蓋的主題]

## Part 1: Vocabulary (8 questions)

## Part 2: Phrases (8 questions)

## Part 3: Grammar (8 questions)

## Answer Key

## Topics Covered
```

### Step 4-7: 批改與更新

同主題測驗的 Step 3-7，但額外記錄涵蓋的主題

---

## 🎯 任務三：查看學習狀態

當用戶詢問學習進度時：

1. 讀取 `progress.md`
2. 彙整各主題狀態
3. 建議下一步學習方向（優先弱項主題）

---

## 📋 注意事項

1. **主題測驗**：每份約 30 題，分 Vocabulary (1-10)、Phrase (11-20)、Grammar (21-30)
2. **混合測驗**：每份 20-25 題，平均分配三種題型
3. **狀態判定**：
   - ⬜ 未開始：測驗次數 = 0
   - 🟡 進行中：有測驗但平均分 < 85%
   - 🟢 已掌握：平均分 ≥ 85%
4. **錯誤追蹤**：區分單字/片語/文法題的錯誤
