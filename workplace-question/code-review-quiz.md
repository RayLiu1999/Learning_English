## Vocabulary Questions 單字題 (1-10)

- D) Login Get The Message

**3.** What is "refactoring"?

- C) Smelly code
- D) Aromatic programming

**7.** What is "peer review"?
- A) Looking at peers
- B) Code evaluation by colleagues
- C) Peer pressure
- D) Review of coworkers

**8.** "Blocking comment" means:
- A) Blocked communication
- B) Issue preventing code merge
- C) Comment blocker
- D) Blocking user comments

**9.** What does "idiomatic" code mean?
- A) Idiotic code
- B) Code following language best practices
- C) Code with idioms

**10.** "Edge case" refers to:
- A) Code at the edge
- B) Unusual or extreme input conditions
- C) Sharp corners in code
