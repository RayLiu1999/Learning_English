# Software Engineering Quiz — 軟體工程英文練習題

## Instructions 說明
每題選擇最合適的答案。題目涵蓋軟體工程相關的單字、片語和文法應用。

## Vocabulary Questions 單字題 (1-10)

**1.** What is "debugging"?
- A) Removing insects
- B) Finding and fixing errors in code
- C) Testing software
- D) Writing documentation

**2.** A "repository" in software context is:
- A) Storage place for code and files
- B) Report generator
- C) Database table
- D) Software company

**3.** What does "deployment" mean?
- A) Using employees
- B) Moving software to production environment
- C) Planning a project
- D) Designing user interface

**4.** "Scalability" refers to:
- A) Weighing something
- B) Ability to handle increased load
- C) Climbing scale
- D) Reducing size

**5.** What is "refactoring"?
- A) Building again
- B) Improving code structure without changing functionality
- C) Adding new features
- D) Creating factors

**6.** A "bug" in software is:
- A) An insect
- B) An error or defect in code
- C) A feature request
- D) Security issue

**7.** What does "API" stand for?
- A) Application Programming Interface
- B) Automatic Program Installation
- C) Advanced Programming Instructions
- D) Application Performance Index

**8.** "Version control" is:
- A) Controlling software versions
- B) System for tracking code changes
- C) Checking version numbers
- D) Managing different releases

**9.** What is a "framework"?
- A) Picture frame
- B) Foundation structure for developing applications
- C) Work schedule
- D) Team structure

**10.** "Integration" means:
- A) Making things whole
- B) Combining different components/systems
- C) Mathematics calculation
- D) Company merger
