# Workplace English大全

本套件幫你系統化學習職場英文：常用單字、片語、句型與情境例句。可依情境快速查找，也可逐步精進。

## 你會得到什麼
- 分情境的大全（Meetings、Emails、Presentations、Interviews、Negotiations...會持續擴充）
- 每個情境都包含：
  - Core Vocabulary（核心字彙與片語，含中文解釋與常見搭配）
  - Useful Patterns（可直接套用的句型）
  - Context Examples（依子情境分類的例句，含中譯）
- 迷你任務（Mini Drills）幫你輸入產出

## 使用建議
1. 先掃過 Useful Patterns，抓住固定句型。
2. 用 Context Examples 以情境感受語氣差異。
3. 用 Mini Drills 做 5 分鐘口說/寫作練習。

## Quick Reference — 中英雙語卡
所有情境檔的文末都提供「中英雙語卡」速查版，精選 8–12 條最常用短句，方便快速複誦與應用。建議：
- 先讀 Useful Patterns，接著用雙語卡口說一輪。
- 將常用句加到你的筆記或 Anki。
