你是否在英文職場溝通中感到吃力？本專案系統化地幫你掌握**各種職場情境的英文**——從會議、郵件、簡報，到面試、談判、客服。每個主題都包含**核心詞彙、實用句型、真實對話範例**，讓你快速上手。

## 🎯 兩大學習系統

### 📚 職場情境學習 (`workplace/` + `workplace-question/`)

19 個職場情境的完整學習資料與測驗：

| 類別          | 情境                                     |
| ------------- | ---------------------------------------- |
| 💼 通用溝通   | 會議、電子郵件、簡報、電話視訊、遠端溝通 |
| 👔 職業發展   | 面試、績效回饋、談判協商                 |
| 💬 專業互動   | 團隊合作、客訴客服、銷售會議、小寒暄文化 |
| 📋 專案流程   | 專案管理、時間管理、報告撰寫、訂單合約   |
| 🔧 工程師專區 | 軟體工程、程式碼審查、事故管理           |

### 📝 單字庫練習 (`vocabulary/`)

個人不熟單字庫，支援：

- 按重要程度分級（1-5 分）
- 混合題型測驗（選擇題、填空題、配對題）
- 錯誤追蹤與優先複習

## 🚀 如何使用

### 職場情境學習

1. 閱讀 `workplace/[topic].md` 學習資料
2. 做 `workplace-question/[topic]/quiz-001.md` 測驗
3. 把答案給 AI，AI 會批改並追蹤進度

### 單字練習

1. 新增不熟單字到 `vocabulary/words.md`
2. 請 AI 產生測驗
3. 做完把答案給 AI，AI 會批改並追蹤錯誤

## ⭐ 使用技巧

1. **不要死背** — 透過實際情境與對話範例來學
2. **主動使用** — 每學一個句型，就在工作中試著用
3. **追蹤錯誤** — 讓 AI 幫你批改，系統會自動追蹤弱點
4. **優先複習** — 系統會優先出錯誤率高的題目
