# 職場英文完全學習指南

你是否在英文職場溝通中感到吃力？本專案系統化地幫你掌握**各種職場情境的英文**——從會議、郵件、簡報，到面試、談判、客服。每個主題都包含**核心詞彙、實用句型、真實對話範例**，讓你快速上手。

## 📚 專案特色

### 📍 涵蓋20+職場情境
每個情境都有詳細的學習資料：
- **核心詞彙 (Core Vocabulary)** — 關鍵單字與片語，含中文解釋與常見搭配
- **實用句型 (Useful Patterns)** — 可直接套用的句型與表達方式
- **情境對話 (Context Examples)** — 依子情境分類的例句，含中英對照
- **雙語快速卡 (Quick Reference)** — 精選8–12條最常用短句，方便記憶與複誦
- **練習題 (Mini Quiz)** — 測驗你的理解程度

## 🚀 如何使用

### 方案1：快速上手（5分鐘）
1. 找到你需要的情境文件（例如 `meetings.md`）
2. 掃過「實用句型 (Useful Patterns)」抓住固定句型
3. 用「雙語快速卡」快速記憶常用短句

### 方案2：深度學習（20–30分鐘）
1. 先讀 **Core Vocabulary**，理解關鍵詞彙的用法與搭配
2. 用 **Useful Patterns** 學習整句表達
3. 通過 **Context Examples** 感受情境中的語氣與細節
4. 用 **Quick Reference** 複誦常用句
5. 完成 **Mini Quiz** 檢驗理解

### 方案3：系統提升（逐步進階）
1. 選擇你工作中最常遇到的3–5個情境
2. 每週深度學習1個情境（Core Vocabulary + Context Examples）
3. 用「Quick Reference」每天複誦一次
4. 在實際工作中套用，記錄自己想到但說不出的表達
5. 逐步擴展到其他情境

## ⭐ 使用這個資源的最佳方式

1. **不要死背** — 透過實際情境與對話範例來學
2. **主動使用** — 每學一個句型，就在接下來的工作中試著用
3. **錄音自己** — 對著手機或電腦複誦「Quick Reference」，聽自己的發音

5. **複習最常用的** — 前20%的表達會用在80%的情況，優先掌握核心短句
